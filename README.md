# TuringDB Visualizer

[![Node.js CI](https://github.com/turing-db/turingdb-visualizer/actions/workflows/ci.yml/badge.svg)](https://github.com/turing-db/turingdb-visualizer/actions/workflows/ci.yml)

- Run the development server:
```bash
npm i
npm run dev

# or
# TURING_FRONTEND_PORT=1234 TURING_API_PORT=6666 npm run dev
```

- Run the production server:
```bash
git checkout prod
npm i
npm run prod
# TURING_FRONTEND_PORT=1234 TURING_API_PORT=6666 npm run prod
```
